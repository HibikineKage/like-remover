# like-remover
ほかの人がいいねしたツイートをTLから消し去ります！

# How to use
* [Like Remover - Chrome ウェブストア](https://chrome.google.com/webstore/detail/like-remover/hkppmnnjafpbeafbaibhkpofjjbiegga?hl=ja) からインストール
* Web版公式ツイッターを開く
* ほかの人のいいねが出なくなります！
